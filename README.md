# What is this fork

This fork, may not be much of a fork, but an alternative version of the local TV to TV installer inside TizenBrewInstaller. It only contains that, so you cannot use this repo/fork to install via PC. <br>
Note: this is not "TizenBrew", it is "TizenBrewInstaller".

## Goals
 * Seperate and remove the PC portion of the project. This is  local-TV-to-TV-installer  app.
 * Simplify the .wgt app portion of the project, and only use a few files in `client_simplified` folder. 
 * Remove bloat. Stay vanilla. 
 * Easy to use, easy to understand, nice on the eyes. Same goes for code.
 * TV mouse and keyboard support.
 * Target 2020 Tizen +5.5 and later versions. Chromium 69 / M69.
